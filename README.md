## Dependencies:

- Node.js
- Express
- ElephantSQL
- Morgan
- Body-Parser
- Axios
- Userfront
- Font-Awesome
- Sass
- React Player
- React

## Getting Started:

- Install dependencies with npm install on both frontend and backend.
- **Run both client and server**:
- -- Running Webpack Development Server: npm start
- -- Running Webpack Development Client: npm start
